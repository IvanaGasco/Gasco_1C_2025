void app_main ( void  )

Función principal del programa.

Inicializa los módulos necesarios:

  • LEDs
  • Switches
  • Sensor analogico
  • Sensor uht11
  • Puerto serie (UART_PC)

Crea la tarea de Mediones de Temperatura y humedad (MedirHumedad_y_Temperatura) y medir radiacion (MedirRadiacion), donde ademas cada tarea enciende un led si es necesario

Configura:

  • Los temporizadores para medir las variables pertinentes y comunicar por uart según los tiempos del enunciado
